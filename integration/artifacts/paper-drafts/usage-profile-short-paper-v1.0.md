# 用途说明：短论文 v1.0

## 定位

本文适合作为一篇短论文或课程论文的人工审阅稿使用。

最稳妥的题目方向是：面向应急救援与军事职业教育的虚拟训练系统辅助价值、场景构建与评价边界。

## 适合怎么用

- 用作综述型、论述型短论文。
- 重点讨论“为什么需要虚拟训练”“虚拟场景如何服务训练任务”“当前证据能支持到哪里”。
- 保持文献分析口径，不把文章写成系统实现报告或实验论文。

## 核心论点

- 虚拟训练系统是训练体系中的辅助和增强手段。
- 它更适合支持训练准备、流程熟悉、典型场景演练、交互操作和评价反馈。
- 虚拟场景构建应从“像不像”转向“是否支撑训练任务”。
- 当前证据不足以证明真实任务能力、协同表现或长期保持效果已经提升。

## 需要人工审阅的点

- 题目、摘要、结论是否匹配投稿或课程要求。
- 参考文献格式是否符合目标规范。
- 是否需要补充更明确的研究问题或贡献表述。
- 是否需要把军事职业教育或应急救援中的某一类场景收窄为主线。

## 不建议怎么用

- 不建议作为最终发表稿直接提交。
- 不建议宣称训练效果已被实验证明。
- 不建议宣称虚拟训练可以替代真实训练或实装操作。
- 不建议把本稿包装成生产级系统、完整 RAG 系统或平台验收材料。
