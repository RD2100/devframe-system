# 参考文献人工核对清单 v1.0

审计对象：

- `integration/artifacts/paper-drafts/local-paper-rag-clean-manuscript-v1.0.md`

## 当前状态

v1.0 已完成自动化层面的引用顺序修正和参考文献重排。剩余工作不是继续自动改格式，而是人工核对元数据。

## 必须人工确认的问题

1. 目标用途是否采用 GB/T 7714 顺序编码制。
2. 正文引用是否需要改成上标格式。
3. 每条文献的作者、题名、期刊名、年份、卷号、期号、页码是否准确。
4. DOI 是否存在且准确。
5. “文章编号”是否应保留。
6. 学位论文 `[6]` 是否应补充保存地点和学校所在地。
7. 正文中每处引用是否确实支撑对应论断。

## 今晚交付建议

如果今晚目标是“正式审阅稿包”，这些问题可以作为人工核对清单随包交付，不必阻塞 v1.0。

## 边界

本清单未联网、未读原始 PDF、未读 Obsidian 原文、未调用外部数据库。它不能替代人工文献核验。
