# PARENT_LOCAL_PAPER_RAG_CLEAN_MANUSCRIPT_V1_0_A1

## Goal

Produce a v1.0 human-review manuscript package from the v0.9 baseline without
expanding runtime scope.

## Scope

- Update the reviewer-facing manuscript to v1.0.
- Normalize sequential citation order and adjacent citation groups.
- Preserve cautious evidence boundaries.
- Add reference audit, human-check list, usage profiles, and claim boundaries.
- Produce a DOCX, Markdown copy, handoff ZIP, and parent closeout evidence.

## Forbidden

- Do not read raw PDFs, Obsidian note bodies, raw chunks, source paths, vectors,
  FAISS binaries, WriteLab payloads, Zotero keys, browser/CDP/cloud/MiniApp
  runtime content, external RAG services, or cloud vector DB artifacts.
- Do not claim paper-quality acceptance, training-effect acceptance, production
  readiness, broad RAG readiness, whole-vault readiness, or RuntimeAuthorization.

## Expected Verification

- Hash DOCX, Markdown, and ZIP.
- Confirm DOCX OpenXML is readable.
- Confirm ZIP entries are exact.
- Confirm key cautious-boundary phrases remain present.
- Confirm internal workflow markers and source markers are absent from the
  manuscript.
- Confirm `CURRENT_DELIVERY.md` points to v1.0.
